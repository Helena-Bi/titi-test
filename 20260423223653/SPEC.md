# TITI 人格测试 · 规格说明书

## 1. 项目概述

- **项目名称**: TITI 人格类型指标（Totally Inaccurate Test Indicator）
- **项目类型**: 纯娱乐恶搞人格测试单页应用
- **核心功能**: 通过12道选择题，根据用户对人格测试的态度和行为，反向给出荒诞的"测试人格"标签
- **目标用户**: 社交网络用户，追求娱乐效果

## 2. 视觉规格

### 2.1 整体风格
- **设计语言**: 复古像素风 + 现代极简主义的混搭
- **色彩方案**:
  - 主背景: `#1a1a2e` (深紫蓝)
  - 卡片背景: `#16213e` (深蓝)
  - 强调色: `#e94560` (玫红)
  - 次要色: `#0f3460` (藏青)
  - 文字色: `#eaeaea` (浅灰白)
  - 选项高亮: `#533483` (紫色)
- **字体**: 系统无衬线字体栈（-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif）
- **动效**: 
  - 选项点击: scale(0.98) + 背景色渐变，200ms ease
  - 页面切换: fadeIn 300ms
  - 结果 emoji: 弹跳入场动画

### 2.2 布局结构
- **起始页**: 居中卡片，标题 + 引语 + 开始按钮
- **答题页**: 顶部进度 → 题目 → 选项列表 → 底部返回按钮
- **结果页**: emoji大字 → 名称 → 金句 → 描述 → 彩蛋 → 重测按钮

### 2.3 响应式策略
- 移动端优先，最大宽度 480px
- 选项按钮全宽，便于点击
- 字号适中，保证可读性

## 3. 功能规格

### 3.1 页面流程
```
[起始页] → [答题页 1/12] → ... → [答题页 12/12] → [结果页]
              ↑↓ 可回退
```

### 3.2 起始页
- 显示测试名称 "TITI 人格测试"
- 显示引语 "你是哪种人格，取决于你怎么被测"
- "开始测试" 按钮

### 3.3 答题页
- 顶部: 当前题号 "X/12"
- 中部: 题目文本 + 3个选项按钮
- 选项: A/B/C 字母标签 + 选项文本
- 点击选项: 
  1. 高亮选中项
  2. 延迟 400ms 自动跳转下一题
  3. 记录答案
- 底部: "← 上一题" 文字按钮（第一题不显示）

### 3.4 结果页
- 大 emoji 图标 (80px)
- 人格名称 (2rem)
- 金句引用 (1.2rem, 斜体)
- 完整描述段落
- 彩蛋文字: "本测试准确度：0%，你信了吗？"
- "再测一次" 按钮

### 3.5 计分系统
- 15种结局类型
- 12道题，每题3选项
- 选项映射到多个结局，每次选择为对应结局+1分
- 最高分获胜；平局按预设优先级输出

## 4. 数据规格

### 4.1 题目数据
- 12道单选题，每题3个选项
- 存储于 JS 数组 `questions[]`

### 4.2 结果数据
- 15种结局，含 emoji、name、quote、desc
- 存储于 JS 对象 `results{}`

### 4.3 分数映射
- 12×3 的映射表 `scoreMap[][]`
- 存储每个选项对应的结局列表

## 5. 验收标准

- [ ] 起始页正常显示，有开始按钮
- [ ] 点击开始进入第1题
- [ ] 每题3个选项，点击后高亮并自动前进
- [ ] 进度显示正确 (1/12 ~ 12/12)
- [ ] 可通过"上一题"返回并修改选择
- [ ] 第12题选择后直接显示结果（无加载动画）
- [ ] 结果页显示 emoji、名称、金句、描述
- [ ] 彩蛋文字显示
- [ ] "再测一次"按钮可重置并返回第1题
- [ ] 移动端布局正常，触摸交互流畅
- [ ] 无外部依赖，纯离线可用
